1. Go to end of file: Shift+ G
2. Go to beginning of file: gg
3. Go to specific line number: :"lineno"
3. Undo: u
4. Redo: Ctrl + r
5. (A). Tab multiple lines: 
  a. Enter Visual Mode (shift + v)
  b. select text using arrow keys
  c. shift + >
  
  (B) Untab multiple lines: 
  a. Enter Visual Mode (shift + v)
  b. select text using arrow keys
  c. shift + <

6. Search for text: /"search text"
7. show line numbers: set nu
8. Hide line numbers: set nu!
9. Go to end of line: $
10. Go to start of line: ^
11. Delete lines from Visual mode:
Shift+V to move to visual mode then select lines and press "d" to delete selected lines.
12. Copy paste in vi editor: press i-> ctrl+shift+V
Or
Use :set paste to disable automated indenting etc.
